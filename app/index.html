O mnie

Cześć! Jestem Karolina i mieszkam w Krakowie. Tworzeniem stron zaczęłam interesować się pod koniec 2017 roku i od tamtej pory rozwijam swoje umiejętności w tej dziedzinie. Na początku samodzielnie rozwijałam swoje umiejętności, ale później zdecydowałam się na udział w półrocznym bootcampie Kodilli. Obecnie, już po kursie, samodzielnie rozwijam swoje umiejętności poprzez czytanie książek związanych z tematem, ze szczególnym naciskiem na JavaScript - aby lepiej zgłębić jego tajemnice oraz robieniem małych projektów. W lutym podjęłam staż na stanowisku Junior Front-end Developera. Efekty mojej dotychczasowej pracy można zobaczyć na tej stronie. Zapraszam również do odwiedzenia mojego Githuba.

Projekty

Landing Page

HTML, Sass, JS, Gulp

Projekt responsywnej landing page z wykorzystaniem HTML, Sass, JavaScript oraz task managera Gulp. W projekcie wykonałam dwupoziomowe menu, stylowanie elementów odtwarzacza mp3 oraz dodałam karuzelę wykorzystując bibliotekę slick. Wykonałam również podstawową walidację formularza oraz zapisywanie maili do bazy danych Firebase.

Theme - Shopware

HTML, Sass, JS, Gulp, Shopware

Theme dla sklepu internetowego z wykorzystaniem platformy ecommerce - Shopware, wykorzystując Sass, jQuery oraz Gulp. Oprócz nadania styli dla strony głównej oraz strony kategorii utworzyłam plugin ze spersonalizowanym elementem „shopping-world”. W projekcie zachowane było podejście RWD.

Smoothie webpage

HTML, Sass, JS, Gulp

Projekt responsywnej strony z wykorzystaniem HTML, Sass, JavaScript oraz task managera Gulp. Wykorzystana była również biblioteka Slick do wykonania karuzeli. Samodzielnie zaprojektowałam wygląd strony, wykonałam podstawową walidację formularza oraz zapisywanie maili do bazy danych Firebase.

Kanban board

HTML, CSS, jQuery

Projekt tablicy z kolumnami, które zawierają karteczki. Karteczki można przenosić pomiędzy kolumnami, edytować, zmieniać im kolor. Projekt napisany z wykorzystaniem składni jQuery i programowania obiektowego.

Random quote machine

HTML, CSS, jQuery, API, ajax

Aplikacja losująca cytaty pobierane z zewnętrznego API. Cytaty można udostępnić na twitterze - wybrano tylko te, które mają do 140 znaków.

Country Search Engine

HTML, CSS, jQuery, API, ajax

Wyszukiwarka krajów napisana w jQuery. Informacje o krajach są pobierane z zewnętrznego API.

To do list

HTML, Sass, JS

Projekt listy zadań wykonanej w JavaScripcie. Zadaniom można nadać priorytet. Listę zadań można sortować po nazwie statusie wykonania. W projekcie wykonana zostałą paginacja, dzięki której można wyświetlać ograniczoną ilość zadań na stronę. Zaqdania można usuwać i zapisują się one w pamięci lokalnej.

Stopwatch

HTML, CSS, JS(ES6)

Projekt stopera napisany z wykorzystaniem składni ES6. Stoper posiada funkcje, które go uruchamiają, zatrzymują, resetują, a także dodają czas do listy i czyszczą tą listę.
